# 📋 CRUD 1

Proyecto realizado con HTML y JavaScript puro.

## Funcionalidades

- Agregar clientes.
- Mostrar la lista de clientes.
- Editar clientes.
- Actualizar la información de un cliente.
- Eliminar clientes.
- Validar que nombre, apellido y edad sean obligatorios.
- Restablecer automáticamente el formulario después de guardar o editar.

## Tecnologías utilizadas

- HTML5
- JavaScript (ES6)

## Lo que aprendí

Durante este proyecto practiqué:

- Manipulación del DOM.
- Variables y funciones.
- Objetos y arrays.
- Bucles `for`.
- Eventos (`addEventListener`).
- `querySelector` y `querySelectorAll`.
- Uso de `dataset`.
- CRUD (Crear, Leer, Actualizar y Eliminar).
- Validación de formularios.
- Actualización dinámica del contenido del DOM.
